Projects

Suitability Analysis for "Underutilized" Agricultural Land in the Annapolis County

Through partnership with the Annapolis County Federation of Agriculture (ACFA) and the Centre of Geographic Sciences (COGS), a suitability analysis of the Annapolis County was done to determine the location of land suitable for agriculture. Next, the Nova Scotia Agricultural Land Inventory Project data from 2018 was used to compare existing farmland to the areas suitable for farmland. The result was the identification of "underutilized" farmland in the County. This work was done primarily in ArcGIS Pro using ModelBuilder. The final results were communicated to stakeholders using an ESRI StoryMap.

Interpolating Time-Series Climate Data Using ArcPy

ClimateData.ca provides detailed historical and projected climate data, ranging from 1950 to 1951. Data is made available on the website in superimposed time-series point data. This project used ArcPy to interpolate climate data values across a study area and create an individual raster for each of the 151 years of data.
